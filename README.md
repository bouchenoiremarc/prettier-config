## Installation

#### Yarn

```bash
yarn add -D prettier @bouchenoiremarc/prettier-config
```

#### npm

```bash
npm install --save-dev prettier @bouchenoiremarc/prettier-config
```

## Usage

Add it to `package.json`.

```json
{
  "prettier": "@bouchenoiremarc/prettier-config"
}
```
